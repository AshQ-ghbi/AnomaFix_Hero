# 🌟 Introduction

Modern cities generate thousands of civic issues every day—from potholes and overflowing garbage bins to broken streetlights and water leakages. While many municipalities provide complaint portals, these systems are often fragmented, difficult to use, slow to respond, and lack transparency.

Citizens frequently submit complaints without receiving updates, while local authorities struggle with duplicate reports, poor categorization, and inefficient resource allocation.

**AnomaFix** is designed to bridge this gap.

AnomaFix is an AI-powered civic issue management platform that empowers citizens to actively participate in improving their communities by making issue reporting simple, transparent, and intelligent.

By combining Artificial Intelligence, geolocation, image recognition, community verification, and real-time tracking, AnomaFix transforms traditional complaint systems into a collaborative digital ecosystem.

---

# 🚨 The Problem

Across cities and towns, citizens commonly face challenges such as:

- Damaged roads and potholes
- Water leakages
- Garbage accumulation
- Broken streetlights
- Blocked drainage systems
- Illegal dumping
- Fallen trees
- Traffic signal failures
- Public property damage

Current reporting systems often suffer from:

❌ Manual categorization

❌ Duplicate complaints

❌ No real-time updates

❌ Limited transparency

❌ Poor citizen engagement

❌ Slow response times

❌ Lack of prioritization

❌ No predictive maintenance

As a result, important issues remain unresolved while public trust declines.

---

# 💡 Our Solution

AnomaFix provides an intelligent digital platform where citizens can report issues within seconds.

Instead of simply collecting complaints, the platform uses Artificial Intelligence to:

- Understand the reported issue
- Analyze uploaded images
- Determine severity
- Detect duplicates
- Recommend the responsible department
- Prioritize issues
- Track progress
- Notify citizens automatically

The platform also enables community participation through verification, voting, and collaboration, creating a transparent workflow from reporting to resolution.

# 🚀 Installation Guide

Follow these steps to set up AnomaFix on your local machine.

---

## 📋 Prerequisites

Ensure the following software is installed before starting.

- Node.js (v18 or above)
- npm or yarn
- Git
- Supabase Project
- Google Gemini API Key

Verify installation:

```bash
node -v
npm -v
git --version
```

---

## 📥 Clone Repository

```bash
git clone https://github.com/AshQ-ghbi/AnomaFix_Hero.git
```

Navigate into the project.

```bash
cd AnomaFix_Hero
```

---

## 📦 Install Dependencies

Using npm:

```bash
npm install
```

or using yarn:

```bash
yarn
```

---

## ⚙️ Environment Variables

Create a `.env.local` file in the root directory.

```env
VITE_SUPABASE_URL=your_supabase_project_url

VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

VITE_GEMINI_API_KEY=your_gemini_api_key

VITE_APP_NAME=AnomaFix

VITE_MAP_PROVIDER=OpenStreetMap
```

Never commit this file to GitHub.

Instead, include a `.env.example` file for contributors.

---

Start Vite.

npm run dev

Open

http://localhost:5173

The application automatically reloads whenever changes are saved.
